원인 첫 번째

 ​구구단을 출력하기 위해서는 정수를 입력받아야한다. Scanner 에서 정수를 입력받는 메서드는 nextInt()이다.

하지만 코드에서는 문자열과 관련된 next()로 입력받고 있으므로 자료형이 일치하지 않으므로 문제가 발생하기 때문에 수정해줘야한다.


원인 두​ 번째

 입력받는 값을 input이라는 변수에 저장하고 있고 그것으로 입력받는 값이 2 이상이고 9 이하인지를 if문의 조건으로 설정하여 판별해야한다. 하지만 코드에서는 if의 조건이 input <= 9로 되어있다. 이것은 9보다 작거나 같은 값은 모두 성립된다. 즉 2보다 더 작은 값이 들어와도 성립된다. 하지만 의도한 것은 2이상이고 9이하이므로 수정해줘야한다.



원인 세​ 번째

 System.out.printf("%d * %d = %d\n", input, num); 문장에서 지시자 %d는 정수를 뜻하고 3개를 사용했다. 이것은 3개의 정수를 지시자 위치에 대입하여 사용하겠다는 것이다. 하지만 변수는 2개만 대입되고 있으므로 마지막 %d에서 오류가 발생한다.



원인 네 번째

 현재 반복문 while은 조건을 true로 설정하여 무한반복이 되고 있다. 2 ~ 9 사이의 숫자가 입력될때까지 반복하는것은 의도한것으로 문제가 아니다. 하지만 의도한것은 2 ~ 9사이의 정수가 입력되면 해당 구구단을 출력하고 종료하는것이므로 구구단을 출력하는 for문이 종료된 후에 break;를 추가하여 while문을 종료해야한다.​



원인 첫 번째에 대한 해결방안

...

int input = sc.nextInt();

...

​정수를 입력받아야 하므로 nextInt()로 변경해야한다.

​

원인 두 번째에 대한 해결방안

...

if(2 <= input && input <= 9) {
// 구구단 출력 for문
}​
...
if(input <= 9)​을 if(2 <= input && input <= 9) 로 변경해줘야한다. &&를 사용한 이유는 두 개의 조건이 모두 참일 경우에만 실행해야하기 때문이다.

원인 세 번째에 대한 ​해결방안
...
for(int num = 1; num <= 9; num++) {
	System.out.printf("%d * %d = %d\n", input, num, input * num);
}
...
input * num 을 추가한다.

원인 네 번째에 대한 ​해결방안
...
if(2 <= input && input <= 9) {
	for(int num = 1; num <= 9; num++) {
		System.out.printf("%d * %d = %d\n", input, num, input * num);
	}
	break;
}
...

for문이 종료된 후에 break문을 추가하여 무한반복 while문을 종료한다​
